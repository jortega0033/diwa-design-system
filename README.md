# Diwa Design System Monorepo

This repository contains the Diwa component library and framework wrapper packages:

- `diwa-components` — core Stencil web components package (`@diwa/components`)
- `diwa-components-react` — React wrappers (`@diwa/components-react`)
- `diwa-components-vue` — Vue wrappers (`@diwa/components-vue`)
- `diwa-components-angular` — Angular wrappers (`@diwa/components-angular`)

The wrappers are generated from the core Stencil configuration in `diwa-components/stencil.config.ts`.
